Performances of Girolle.

Introduction

The performance of Girolle is a really important matter. The lib is made to be fast and to be able to handle a lot of request.

Cost of the serialization & deserialization

The cost of the serialization and deserialization is really low. The lib use serde_json to serialize and deserialize the data.

The average overhead of the serialization and deserialization is around 39 ns per call.
